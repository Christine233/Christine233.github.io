C Primer Plus - 字符串和格式化输入/输出 总结

2018.07.09 一白 C  热度 ℃

字符串简介

  • C语言没有专门用于储存字符串的变量类型，字符串都被存储在char类型的数组中。
  • ‘\0’：空字符（ASCII码值为0），用来标记字符串的结束。
  • 数组：同类型数据元素的有序序列

    注意字符串结尾要加’\0’

引号

  • 双引号: 告知编译器它括起来的是字符串
  • 单引号： 用于标识单个字符

strlen()函数

sizeof()，以字节为单位给出对象的大小
strlen() 函数给出字符串中的字符长度

常量

#define NAME value
value 可以是数值，字符，字符串
const关键字：限定一个变量为只读

printf()和scanf()的*修饰符

printf()：用 * 修饰符代替字段宽度。可用于变宽输出字段
scanf()：把 * 放在%和转换字符之间，会使scanf()跳过相应的输出项。可用于读取文件中特定列的内容。
