C Primer Plus - 第六章 用关系运算符和表达式比较大小 总结

2018.08.30 一白 C  热度 ℃

表达式相关

关系表达式值为真 –> 该表达式的值是 1
关系表达式值为假 –> 该表达式的值是 0

一般而言，所有的 非零值 都视为真，只有 0 被视为假

运算符

运算符优先级

运算符（优先级从高至低） 结合律
() 左 -> 右
-, +, ++, –, sizeof 右 -> 左
*, /, % 左 -> 右
+, - 左 -> 右
<, >, <=, >= 左 -> 右
==, != 左 -> 右
= 右 -> 左

优先级：算术运算符 > 关系运算符 > 赋值运算符

字符在内部是以整数形式储存的

逗号运算符

把两个表达式连接成一个表达式，并保证最左边的表达式最先求值。逗号运算符通常在 for 循环头的表达式中用于包含更多的信息。整合逗号表达式的值是逗号右侧表达式的值。
