    2.问题描述如下

    解决办法：不用 win cmd 改用 git bash

    此时我又发现自己右键没有 打开git bash 喵喵喵？？？
    据说是因为我把 git 装在了d盘…

    于是开始进行 将 git bash 添加进注册表，并可以在文件夹空白处右键菜单添加 Git Bash Here 的操作：

    • win + r 输入 regedit 打开注册表编辑器
    • 进入 HKEY_CLASSES_ROOT\Directory\Background\shell
    • 在shell下新建项并命名为 Git Bash Here
    • 在 Git Bash Here 目录下新建项并命名为 Command ，其值为"D:\software\manong\GitHub\Git\git-bash.exe"，即git-bash.exe 所在的完整路径（自己加自己的，这里是参考），注意加引号。如下图所示：
    • 保存后，随便找个文件夹空白处右键，即会发现新加了 Git Bash Here 快捷键。
