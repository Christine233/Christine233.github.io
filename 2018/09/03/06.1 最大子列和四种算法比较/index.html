最大子列和四种算法

2018.09.03 一白 算法  热度 ℃

01.暴力解决 —— 枚举

遍历所有可能性，选出最大子列
算法复杂度：$ O(N^2) $

每一个复杂度为 $ O(N^2) $ 的算法都要看能不能变成 $ O(N(logN)) $

02.分而治之

  1. 把数组从中间一分为二
  2. 递归的去解决左右两边的问题

算法复杂度：$ O(N\log N) $

推导：
$ T(1) = O(1) $
$ T(N) $
$ = 2T(N/2) + cN $
$ = 2[2T(N/2^2) + cN/2] + cN $
$ = 2^kO(1) + ckN $
$ = N\log N $
// ($ N/2^k = 1$)

03.在线处理

在线，指每输入一个数，就进行即时的处理，在任何一个地方终止输入，算法都能给出正确的、当前的解
思路：从前加到后，保留正数
算法复杂度：$ O(N) $
